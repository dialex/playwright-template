# E2E test repo template (with Playwright)

## 📦 Install

Prerequisites: latest version of Node (you can use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) to install it, ie. `nvm install`)

```sh
npm run install-deps
```

## ✅ Test

```sh
npm test
```

## 🏎 Run

```sh
npm run debug
npm run record
```
